# Oidea 協作平台 — 需求追蹤文件

> 最後更新：2026-04-20

## 概述

Oidea 是一個整合即時通訊、專案管理、會議系統、白板協作編輯的全平台應用。
參考產品：JANDI、Slack、Affine。

**技術棧：** Flutter (Web/iOS/Android) + NestJS + PostgreSQL + Redis + Docker Compose

---

## 非功能需求

| -------- | ----------------------------- | ------------------------- |
| 效能     | 訊息傳遞延遲                  | < 200ms                   |
| 效能     | 白板操作延遲                  | < 100ms (本地)            |
| 效能     | 同時線上使用者 (單工作空間)  | 500+                      |
| 效能     | 視訊會議參與者               | 25人                      |
| 可靠性   | 系統可用率                    | 99.9%                     |
| 安全性   | 資料傳輸加密                  | TLS 1.3                   |
| 安全性   | 密碼儲存                      | bcrypt (salt rounds: 12)  |
| 相容性   | Web 瀏覽器                    | Chrome/Safari/Firefox 最新兩版 |
| 相容性   | iOS                           | iOS 15+                   |
| 相容性   | Android                       | Android 10+               |
