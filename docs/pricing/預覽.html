📋 典型流程

  1. 客戶詢問 → 打開「💎 產品價目表」確認方案
  2. 複製「🧮 報價單範例」→ 改為客戶名稱
  3. 成交 → 「📋 客戶主檔」加一列
  4. 每方案 → 「📦 服務項目」加一列
  5. 收款 → 「💵 收款紀錄」加一列
