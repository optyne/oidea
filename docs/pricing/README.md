# 產品價目與客戶服務工作簿

> 適用對象：以官網維護、企業 Email、別名服務等為主要營收的小型工作室／個人接案者
> 最後更新：2026-04-20

此資料夾提供一套可直接匯入 Excel / Google 試算表的 CSV 檔，涵蓋 6 張工作表，涵蓋從**報價 → 成交 → 收款**的完整流程。

---

## 📁 工作表一覽

| # | 檔名 | 工作表名稱 | 用途 |
|---|------|-----------|------|
| 1 | `01-使用說明.md` | 📘 使用說明 | 工作簿導覽、填寫順序、欄位定義 |
| 2 | `02-產品價目表.csv` | 💎 產品價目表 | 所有標準服務的統一價目與產品代碼 |
| 3 | `03-客戶主檔.csv` | 📋 客戶主檔 | 客戶基本資料、聯絡方式 |
| 4 | `04-服務項目.csv` | 📦 服務項目 | 每位客戶已購買的服務明細（以產品代碼對應 #2） |
| 5 | `05-收款紀錄_2026.csv` | 💵 收款紀錄 2026 | 年度收款流水帳 |
| 6 | `06-報價單範例.csv` | 🧮 報價單範例 | 新客戶報價用範本（複製後改名使用） |

## 🔗 資料關聯

```
產品價目表 (#2)         客戶主檔 (#3)
  │                        │
  │ 產品代碼               │ 客戶編號
  ▼                        ▼
     ┌──────────────┐
     │ 服務項目 (#4) │
     └──────────────┘
            │ 項目編號
            ▼
  收款紀錄 (#5)      報價單 (#6)
```

- **產品代碼** 是 #2 的主鍵，在 #4、#6 引用
- **客戶編號** 是 #3 的主鍵，在 #4、#5、#6 引用
- **項目編號** 是 #4 的主鍵，在 #5 引用

## 🏷️ 產品代碼命名規則

格式：`<類別>-<識別碼>`

| 類別字首 | 說明 |
|---------|------|
| `WEB-` | 網站相關（維護、架設、SSL） |
| `MAIL-` | Email 主方案 |
| `CAP-` | 容量加購 |
| `ALIAS-` | 別名服務 |

識別碼：
- 數字 `01`、`02` 代表方案等級或版本
- 字母縮寫（如 `LITE`/`STD`/`PRO`、`S`/`M`/`L`、`T`）代表規格或類型

新增代碼時請維持格式一致，以便未來做篩選與統計。

## ✅ 建議每月作業

1. **每月初**：檢查 `04-服務項目` 中「到期日」欄位，該月到期者寄出續約通知
2. **收款當下**：在 `05-收款紀錄_2026` 加一列
3. **新客戶**：先在 `03-客戶主檔` 建檔，再從 `06-報價單範例` 複製新工作表
4. **每季末**：用「產品代碼」欄位做樞紐表，檢視哪些方案熱賣

---

## 📝 待使用者決定的設定

以下欄位本範本已預留，但請依實際營運習慣調整：

- **折扣方案欄位**：`04-服務項目.csv` 預留「折扣%」欄，未使用可忽略
- **幣別**：預設新台幣（NTD），如需多幣別請自行增加欄位
- **稅別**：目前以含稅價呈現，若需分離稅額可另加「未稅」「稅額」欄

## 💡 進階技巧

- 匯入 Google 試算表後，對「產品代碼」欄設 **資料驗證 → 下拉式選單**，來源指向 `02-產品價目表` 的 B 欄
- 在 `04-服務項目` 的「單價」欄用 `VLOOKUP` 或 `XLOOKUP` 從價目表自動帶入：
  ```
  =XLOOKUP(D2, '產品價目表'!C:C, '產品價目表'!F:F, "查無")
  ```
- 在 `05-收款紀錄` 的年度總額：`=SUM(E:E)`，月結則搭配 `SUMIFS`
